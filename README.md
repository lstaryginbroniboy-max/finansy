# Финансы — Личный бюджет

Веб-приложение для учёта и планирования личных финансов.

## Возможности

- **Доходы** — учёт источников дохода по категориям
- **Расходы** — учёт трат по категориям с фильтрацией
- **Цели** — накопительные цели с расчётом суммы откладывания (день/неделя/месяц), наградой и фото
- **Статистика** — графики по месяцам, структура доходов и расходов, динамика баланса
- **Настройки** — валюта, тема, категории, экспорт/импорт данных

## Использование

Данные хранятся в `localStorage` браузера — без сервера и регистрации.
